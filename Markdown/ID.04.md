# 0. **Axiom R0** står ensamt och icke-generativt



- Nödvändighet utan innehåll
    
- Ingen process, ingen form, ingen riktning
    
- Endast villkor för att något överhuvudtaget kan ske
    
- Kan inte itereras, kan inte kombineras
    

---

# 1. **Relation**



- Samexistens utan objekt
    
- Skillnad utan separation
    
- Kontakt utan medium
    
- Förutsättning för all senare struktur
    

---

# 2. ⇒ **Differentiering**


- Differentiering utan relationsbrott
    
- Samma relation, flera aspekter
    
- Skillnad utan fragmentering

- Icke-identitet uppstår
    
- Relation kan inte längre vara singulär
    
- Skillnad utan värdering
    
- Första brytningen (inte rörelse)
    

---

# 3. ⇒ **Två poler**


- Ömsesidigt beroende ytterligheter
    
- Ingen absolut referenspunkt
    
- Spänning utan riktning
    
- Polarisering utan kraft
    

---

# 4. ⇒ **Fusion / överlapp (Vesica)**



- Samtidig separation och kontakt
    
- Delat centrum utan sammanslagning
    
- Första geometriska läsbarheten
    
- Relationen blir lokaliserbar
    

---

# 5. ⇒ **Statisk asymmetri (residual)**



- Oförenliga centrumkrav
    
- Ingen perfekt statisk lösning
    
- Obalans utan förlust
    
- Ett måste, inte ett fel
    

---

# 6. ⇒ **Cirkulation / rörelse**



- Asymmetri kräver passage
    
- Rörelse som villkor, inte effekt
    
- Ingen fast punkt möjlig
    
- Tidslös cirkulation
    

---

# 7. ⇒ **Struktur i rörelse = energi**



Rörelse utan struktur är upplösning.  
Struktur utan rörelse är kollaps.


- Energi som relationell aktivitet
    
- Rörelse med bevarad struktur
    
- Ingen magnitud, endast funktion
    
- Stabilitet genom kontinuerlig omfördelning
    

---

# 8. ⇒ **Återkommande mönster**



- Samma struktur i ny bärarform
    
- Ingen ny information
    
- Fraktal bevarandeprincip
    
- Skalning utan förändring
    

---

# 9. ⇒ **Minsta stabila 3D-cell**



- Volym krävs för cirkulation
    
- Minsta slutna bärarstruktur
    
- Alla relationer återkopplade
    
- Kan delas men inte reduceras
